A responsive, static web application for managing shopping lists and inventory with a modern Neo Brutalism design.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Neo Brutalism UI**: High contrast, bold borders, and vibrant colors.
- **Smooth Animations**: View Transitions API integration for app-like navigation.
- **Static Architecture**: HTML5, TailwindCSS (Script), and Vanilla JS.

## Deployment on Vercel

This project is optimized for [Vercel](https://vercel.com).

1. **Push to GitHub/GitLab/Bitbucket**.
2. **Import Project** in Vercel.
3. **Deploy** (No build command required).

### Vercel Configuration
The included `vercel.json` enables:
- **Clean URLs**: `.html` extensions are optional in URLs (served at `/login` instead of `/login.html` if desired, though internal linking uses `.html` for local compatibility).
- **Caching**: Optimized headers for static assets.
